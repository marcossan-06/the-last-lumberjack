# 🌲 The Last Lumberjack

> Videojuego shooter de supervivencia en tercera persona desarrollado en Unity con C#, acompañado de una web complementaria desplegada en producción para permitir una descarga centralizada.

---

## 🎮 Descripción del Proyecto

**The Last Lumberjack** es un videojuego **shooter de supervivencia en 3ª persona** desarrollado con **Unity** utilizando **C#** como lenguaje principal.

El jugador debe sobrevivir enfrentándose a diferentes oleadas de mounstros en el bosque, poniendo a prueba sus reflejos y estrategia.

El proyecto incluye una página web desplegada en producción que funciona como **landing page oficial** para presentar el juego, presentar la historia y facilitar su descarga.

El objetivo es ofrecer una experiencia completa:

- 🎮 Juego multiplataforma  
- 🌐 Web pública en producción  
- 📦 Descarga centralizada desde GitHub o la propia web  

---
